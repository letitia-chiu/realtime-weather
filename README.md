# 即時天氣 (Realtime weather)
React 練習用專案，參考書籍[《從 Hooks 開始，讓你的網頁 React 起來》](https://www.books.com.tw/products/0010870186)

## 版本
- v1.0.0 (2024.01.29) ⭐️ 目前版本

## 功能 (Features)
- 可查看台灣各地即時天氣
- 使用者可設定欲查看之地區
- 支援深色模式
  - 自動隨各地日出/日落時間變更淺色/深色模式
  - 可手動更改淺色/深色模式

## 開發執行環境 (RTE)
[Node.js](https://nodejs.org/) (v20.10.0)
